C# Windows Forms Nedir?

🏠

Windows Forms = bilgisayarında gördüğün butonlu, yazılı programların yapıldığı bir oyun alanı. Sen bir mimar gibisin. Form dediğimiz şey boş bir kağıt. Üzerine butonlar, yazı kutuları, etiketler koyarak bir program inşa ediyorsun. C# ise bu parçalara "ne yapacaklarını" söyleyen dil.

Bir Windows Forms programı 3 şeyden oluşur:

Form (Pencere) — Programın kendisi. Boş bir sayfa gibi düşün.
Kontroller (Araçlar) — Buton, yazı kutusu, etiket gibi parçalar. Formun üzerine koyarsın.
Kod (Olaylar) — Bu parçalara tıklanınca ne olacağını yazan kısım.
Günlük hayattan örnek: Bir hesap makinesi düşün. Ekranı (Label), tuşları (Button), yazı yazdığın alanı (TextBox) var. İşte bunların hepsi birer "kontrol"dür. Windows Forms ile sen de böyle programlar yapabilirsin!

Değişkenler — "Bilgi Saklama Kutuları"

📦

Değişken = üzerinde etiket olan bir kutu. Kutunun içine bir şey koyarsın, sonra lazım olduğunda açıp bakarsın. Kutunun üzerindeki etiket onun adıdır. Kutunun rengi (tipi) ise içine ne tür şey koyabileceğini gösterir.

En Çok Kullanılan 3 Kutu Tipi

TipKutu Rengi (Anlamı)İçine Ne Konur?Örnek
stringPembe kutuYazılar, isimler, şehirler"Fatih", "Ankara"
intMavi kutuTam sayılar (1, 42, -5)42, 0, -10
doubleYeşil kutuOndalıklı sayılar3.14, 68.5
Değişken Nasıl Yazılır? (Syntax)
string sehir;          // 1.Adım: "sehir" adında pembe bir kutu oluştur (boş)
sehir = "Ankara";      // 2.Adım: Kutunun içine "Ankara" yazısını koy
// VEYA tek seferde:
string sehir = "Ankara";  // Hem oluştur hem değeri koy
Altın Kural: Değişkeni kullanmadan önce mutlaka bir değer vermelisin! Boş kutuyu kullanamazsın. string sehir; label1.Text = sehir; yazarsan hata alırsın çünkü kutu boş!
Pembe kutuya sayı koyamazsın! string içine sadece yazı, int içine sadece tam sayı konur. Yanlış tipte şey koymaya çalışırsan program kızar.

Form Üzerindeki Araçlar (Kontroller)

Bunlar formun üzerine sürükleyip bıraktığın hazır parçacıklar. Her birinin kendine has işi var:

Label

Sabit yazı gösterir. Kullanıcı değiştiremez. "Sonuç: 42" gibi.

TextBox

Kullanıcının yazı yazdığı kutu. Kullanıcı buraya ismini, sayıyı vs. yazar.

Button

Tıklanınca bir şey yapan düğme. "Hesapla", "Kaydet" gibi.

ListBox

Liste şeklinde veri gösterir. Alt alta sıralı kayıtlar için.

ComboBox

Açılır liste. Tıklayınca seçenekler aşağı doğru açılır.

CheckBox

İşaret kutusu. "Kabul ediyorum" gibi onay kutuları.

RadioButton

Tek seçim düğmesi. Birkaç seçenekten sadece biri seçilir.

PictureBox

Resim kutusu. Resim göstermek için veya tıklanabilir ikon olarak kullanılır.

ProgressBar

İlerleme çubuğu. %0'dan %100'e dolan bir çubuk.

RichTextBox

Büyük yazı alanı. TextBox'ın çok satırlı, geniş versiyonu.

Olaylar (Events) — "Bir Şey Olduğunda Çalışan Kodlar"

🔔

Olay (Event) = bir şey olduğunda çalan zil. "Butona tıklandı" zili çalınca senin yazdığın kod çalışır. "Form yüklendi" zili çalınca başka kod çalışır. Her zile bağlı bir kod bloğu vardır.

private void kontrolAdi_OlayAdi(object sender, EventArgs e)
{
    // Olay olduğunda çalışacak kodlar buraya yazılır
}

Bu kalıp her olay için aynıdır. Sadece kontrolAdi ve OlayAdi değişir:

Veri Tipi Dönüştürme — "Kutu Değiştirme"

🔄

textBox1.Text her zaman pembe kutudur (string). Ama sen toplama yapmak istiyorsan mavi kutuya (int) çevirmen gerek. Bunun için özel aletler var: Convert.ToInt16() ve .ToString().

Yazıdan → Sayıya

Convert.ToInt16(degisken)
Pembe kutudaki yazıyı mavi kutuya (tam sayı) çevirir.

int.Parse(...) de aynı işi yapar.

Sayıdan → Yazıya

degisken.ToString()
Mavi kutudaki sayıyı pembe kutuya (yazı) çevirir.

Label'a sayı yazdırmak için şart! Label sadece yazı kabul eder.

Neden Dönüştürme Şart?
// YANLIŞ:
int sonuc = textBox1.Text + textBox2.Text;  // "5" + "3" = "53" (yazı birleşir, toplanmaz!)

// DOĞRU:
int sayi1 = Convert.ToInt16(textBox1.Text);  // "5" → 5
int sayi2 = Convert.ToInt16(textBox2.Text);  // "3" → 3
int sonuc = sayi1 + sayi2;                   // 5 + 3 = 8 (sayılar toplanır!)
label1.Text = sonuc.ToString();              // 8 → "8" (label'a yazdırmak için geri çevir)

if-else — "Karar Verme"

🚦

if = "Eğer..." demek. Tıpkı bir trafik ışığı gibi: Eğer yeşilse geç, değilse (else) dur. Programına karar verdirirsin.

Temel Kalıp (Syntax)

if (KOŞUL)
{
    // Koşul doğruysa burası çalışır
}
else
{
    // Koşul yanlışsa burası çalışır
}
KOŞUL yazarken kullanılan işaretler:
== eşit mi? | != eşit değil mi? | > büyük mü? | < küçük mü? | >= büyük veya eşit mi? | <= küçük veya eşit mi? | && VE | || VEYA
Gerçek Örnek: Geçti/Kaldı
double ortalama = vize * 0.3 + final * 0.7;
string durum;

if (ortalama > 59)           // EĞER ortalama 59'dan büyükse...
{
    durum = "Geçti";         // ...geçti yaz
}
else                         // DEĞİLSE...
{
    durum = "Kaldı";         // ...kaldı yaz
}
Sık yapılan hata: if (sayi = 5) yazmayın! Bu değer atar, karşılaştırmaz. Karşılaştırmak için if (sayi == 5) yazın (çift eşittir!). Tek eşittir atama, çift eşittir karşılaştırmadır.

for Döngüsü — "Tekrar Tekrar Yap"

🔁

for döngüsü = bir işi belirli sayıda tekrarlamak. "10 kere zıpla" gibi. Bilgisayara "şu işi şu kadar kere yap" dersin, o da yorulmadan yapar.

Kalıp (Syntax): 3 Parçası Var

for (BAŞLANGIÇ; KOŞUL; ARTIŞ)
{
    // Her turda çalışacak kod
}
// BAŞLANGIÇ: Saymaya nereden başlayalım?     (int i = 1)
// KOŞUL    : Ne zamana kadar devam edelim?    (i <= 5)
// ARTIŞ    : Her turda sayı ne kadar artsın?  (i++)
Örnek: 5 kere "Merhaba" yaz
for (int i = 1; i <= 5; i++)   // i=1'den başla, 5'e kadar, her turda 1 artır
{
    listBox1.Items.Add("Merhaba " + i);
}
// Çıktı:
// Merhaba 1
// Merhaba 2
// Merhaba 3
// Merhaba 4
// Merhaba 5
i++ ne demek? = i'yi 1 artır. i = i + 1 ile aynı şeydir. sayac++ de aynı mantık: sayacın değerini 1 artır.

Diziler (Arrays) — "Sıralı Kutular"

📚

Dizi = yan yana dizilmiş, aynı tipte çok sayıda kutu. Her kutunun bir numarası (index) vardır. İlk kutu 0 numaradır (C#'ta saymaya 0'dan başlanır!).

// Dizi tanımlama:
string[] sehirler = { "Mardin", "Ankara", "İzmir", "Van" };

// Kutulara erişim (index ile):
sehirler[0]  // "Mardin" (ilk kutu, index 0!)
sehirler[1]  // "Ankara"
sehirler[2]  // "İzmir"
sehirler[3]  // "Van"

// Dizinin uzunluğu:
sehirler.Length  // 4 (kaç kutu var?)
İndeks 0'dan başlar! 4 elemanlı dizide son indeks 3'tür. sehirler[4] dersen hata alırsın, çünkü 4. indeks yok!

MessageBox — "Açılır Pencere"

💬

MessageBox = ekranın ortasında beliren küçük bilgi penceresi. Kullanıcıya mesaj göstermek için kullanılır. "Kaydedildi!", "Emin misiniz?" gibi.

Temel Kalıp (Syntax)

MessageBox.Show("Mesaj", "Başlık", Butonlar, Simge);
//                     ↑        ↑         ↑       ↑
//                   Gövde    Pencere   Tamam/  Soru/Ünlem/
//                   yazısı   başlığı   İptal   Bilgi işareti
En Çok Kullanılan MessageBox Kalıpları
// Bilgi mesajı:
MessageBox.Show("Kayıt başarılı!", "Bilgi", MessageBoxButtons.OK, MessageBoxIcon.Information);

// Onay sorusu:
MessageBox.Show("Silmek istediğine emin misin?", "Onay", MessageBoxButtons.YesNo, MessageBoxIcon.Question);

// Değişkenli mesaj:
string isim = "Fatih";
MessageBox.Show("Hoş geldin: " + isim, "Bilgilendirme", MessageBoxButtons.OK, MessageBoxIcon.Information);
// Çıktı: Hoş geldin: Fatih
Buton SeçenekleriSimge Seçenekleri
MessageBoxButtons.OK — Sadece Tamam
MessageBoxButtons.OKCancel — Tamam + İptal
MessageBoxButtons.YesNo — Evet + Hayır		 MessageBoxIcon.Information — Mavi (i)
MessageBoxIcon.Question — Soru işareti
MessageBoxIcon.Warning — Sarı ünlem

Timer — "Otomatik Zamanlayıcı"

⏰

Timer = belirli aralıklarla otomatik olarak kod çalıştıran bir saat. "Her saniye şunu yap" dersin, timer her saniye senin kodunu çalıştırır. Sen bir şey yapmana gerek yok, kendi kendine çalışır!

Dijital Saat Nasıl Yapılır?
int saniye = 0;

private void timer1_Tick(object sender, EventArgs e)  // Her tik'te burası çalışır
{
    saniye++;               // Saniyeyi 1 artır
    label1.Text = saniye.ToString();  // Label'da göster

    if (saniye == 60)       // 60 saniye = 1 dakika
    {
        dakika++;
        saniye = 0;         // Saniyeyi sıfırla, baştan say
    }
}
Timer'ın Interval özelliği kaç milisaniyede bir tik yapacağını belirler. 1000 = 1 saniye. Enabled = true yapmazsan timer çalışmaz!

Random — "Rastgele Sayı Üretme"

🎲

Random = zar atmak gibidir. Bilgisayara "1 ile 10 arasında rastgele bir sayı söyle" dersin, o da sana sürpriz bir sayı verir. Her seferinde farklı olabilir.

Kullanım Kalıbı
// 1.Adım: Zar oluştur
Random uret = new Random();

// 2.Adım: Rastgele sayı al
int sayi = uret.Next(1, 10);   // 1 ile 9 arasında rastgele sayı (10 DAHİL DEĞİL!)
int sayi2 = uret.Next(1, 5);   // 1 ile 4 arasında
int sayi3 = uret.Next(0, dizi.Length);  // Dizi indeksi için: 0'dan uzunluk-1'e kadar
Next(1, 10) → 1 dahil, 10 hariç! Yani 1,2,3,4,5,6,7,8,9 üretir, 10'u ASLA üretmez. İkinci sayı her zaman hariçtir.

Formlar Arası Geçiş — "Pencere Değiştirme"

🪟

Bir programda birden fazla pencere (form) olabilir. Tıpkı tarayıcıda birden fazla sekme açmak gibi. Bir formdan diğerine geçebilir, hatta veri gönderebilirsin.

Form Açma (Temel)
// Form2'yi aç:
Form2 yeniForm = new Form2();   // Yeni bir Form2 oluştur
yeniForm.Show();                 // Onu göster (aç)
this.Hide();                     // Bu formu gizle (Form1'i)
Veri Gönderme (İleri Seviye)
// Form2.cs içinde public değişken tanımla:
public string mesaj;   // "public" = her formdan erişilebilir

// Form1.cs içinde değer ata:
Form2 frm = new Form2();
frm.mesaj = "Merhaba Form2!";   // public değişkene değer yaz
frm.Show();

// Form2_Load olayında kullan:
label1.Text = mesaj;   // Form1'den gelen değeri göster

Dinamik Kontrol — "Kodla Araç Ekleme"

🧱

Forma araçları sürükle-bırak yapmak yerine, kod çalışırken de ekleyebilirsin! Bu sayede program çalışırken yeni butonlar, yazı kutuları oluşturabilirsin. Tıpkı bir inşaat robotu gibi!

Adım Adım: Kodla Buton Ekleme
// 1.Adım: Yeni bir buton oluştur
Button btn = new Button();

// 2.Adım: Özelliklerini ayarla
btn.Name = "dinamikButon";      // Adını ver
btn.Text = "Bana Tıkla!";       // Üstündeki yazı
btn.Width = 120;                 // Genişlik (piksel)
btn.Height = 50;                 // Yükseklik (piksel)
btn.Location = new Point(300, 200);  // Nereye yerleşsin? (x=300, y=200)
btn.BackColor = Color.Aqua;     // Arka plan rengi

// 3.Adım: Formun üzerine ekle (EN ÖNEMLİ ADIM!)
this.Controls.Add(btn);         // Bunu yapmazsan buton görünmez!
